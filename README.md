# 数学小天才 - Math for Kids

一个帮助小朋友学习 10 以内乘法的趣味 web 游戏。

## 在线体验

🔗 **在线地址**: https://math-for-kids-fawn.vercel.app/

![游戏截图](./screenshot.jpg)

## 功能特点

- 🎮 10×10 网格选择
- 🖱️ 支持鼠标/触摸滑动选择
- 🔢 随机 10 以内乘法题目
- ✅ 答对有音效反馈
- 📊 计分系统

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev
```

访问 http://localhost:3000

## 构建部署

```bash
# 生产构建
npm run build

# 生产环境运行
npm run start
```

## 部署到 Vercel

1. 将代码推送到 GitHub
2. 在 Vercel 导入项目
3. 自动构建部署

## 技术栈

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
